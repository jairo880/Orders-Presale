Instalar Io.Js

Configurar proxy para poder trabajar en el sena, y poder instalar con npm

  • Ejecuta esta dos lineas de codido en la consola de CMD

    • npm config set proxy http://proxy2.sena.edu.co:80

    • npm config set https-proxy http://proxy2.sena.edu.co:80

      1. Instalar IoJs: solo descarga el instalador correspondiente a tu sistema e instala como un programa cualquiera, en el siguiente enlace lo puedes hacer https://iojs.org/es/
      1. Para instalar Bower solo debes abrir una consola de CMD y copiar lo siguiente (ya debiste haber instalado IoJs) npm install -g bower, esto instalara bower de manera global
      1. Para instalar las dependencias del proyecto:
        • Ingresas con CMD a la raíz del proyecto, la ruta para wamp es: wamp\www\OrdersPresale\Orders_Presale, luego de encontrarse en esta rruta copias el siguiente comando y ENTER : bower install
